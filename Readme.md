QR Code Generator
========================



## Install NVM
```
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
source ~/.bashrc   
```

## Install NPM

```
nvm install v4.9.1
nvm use v4.9.1
```

python2 needed
```
sudo apt install python2
```

## Build it

```
npm install
npm run build
````

## Run it

```
npm run start
````

## View it

Then view the app:
[http://localhost:3011](http://localhost:3011)
